CanBridge is a cancer detection and support platform designed to assist users

with AI-assisted cancer prediction, emotional support communities, and access

to financial aid resources.



\## Features

\- Cancer prediction using medical images

\- Support forums for patients and caregivers

\- Warrior stories from survivors

\- Financial aid resources

\- Educational cancer information



\## Tech Stack

\- Frontend: Streamlit

\- Backend: FastAPI (planned)

\- Database: PostgreSQL (planned)

\- AI Models: CNN-based deep learning models



\## Run Frontend

pip install -r requirements.txt

streamlit run app/main.py
